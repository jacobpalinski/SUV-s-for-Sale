# SUV-s-for-Sale

Analysis of data from over 1000 SUV's for sale in the Sydney Area scraped using Python's Beautiful Soup module from carsales.com.au. The objective of the analysis was to better understand the factors that influence pricing of SUV's in the broader Sydney area. The analysis addresses the following questions/issues:

* Impact of Brand on Pricing
* Impact of Odoemeter Readings on Pricing
* Depreciation based on Brand

Python Numpy, Pandas, Matplotlib and Seaborn libraries were used for the analysis.
